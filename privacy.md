## Information Collection and Use
Anonymous usage data will only be send when the option “Send Anonymous Usage Info” is enabled in preferences panel.

If you disable the option, no data will be collected to us. (The option can be opt-out, but may need a restart of Themaphack for the option to apply).

When you allow send anonymous usage data, we WILL collect following data anonymous for getting overall statistics of how Themaphack is used:

Themaphack version, operation system, screen resolution, locale and countries (based on IP address).
Name of operations when using Themaphack, such as “launch”, “new window”, “print”, etc, for gathering overall statistics. (We won’t be able to list all detailed operations for a given user. We won’t collect any detail or sensitive data for user’s operation)
How long a user is using Themaphack.
Insensitive preference settings in Themaphack such as theme name, window style, etc.
When error/exception happens, and if you allow send anonymous usage data, we will collect stack trace, related settings, and running status before exception throws to help us find errors. It should not contain any sensitive or personal data.
